App Settings

Accessing the settings of 'The Unlabeler' is straightforward. Tap the gear icon located in the top right corner of the screen to customize your experience:

Basic Settings

  1. Left-Handed Operation : For left-handed users, you can mirror the buttons at the bottom from the right side to the left side by enabling 'left-handed operation'. This makes the app more comfortable and intuitive to use.
  2. Show Zoom Buttons : Show additional zoom buttons on the right side of the screen. You can zoom with two fingers in any case.
  3. Save Camera Photo : If you take a photo directly with the camera, you can choose whether the photo should be saved in the photo library or not. If you do not save it, it will only be used for unlabeling.

Sommelier Package

The following settings are only available if you have purchased the 'Sommelier Package'.

  1. Label Width Options : Customize how the app handles the width of your unlabelled images. There are three different settings:
    • Unlimited: The width of the label is automatically adjusted to be proportional to its height in the photo, without any restrictions.
    • Limited: The width is adjusted to be proportional to the height, but if it exceeds a set limit, the label will be scaled down to fit within that limit.
    • Fixed: The label is scaled to match a specific width exactly as specified. This option is particularly useful for maintaining a consistent presentation in your tasting app, as all labels will have the same width, providing a clear and uniform look.
    You can set the limit for the 'Limited' and 'Fixed' options in the input field, choosing a value between 300 and 5000 pixels to suit your needs.
  2. Filename of the Shared Label : You can specify the filename of the shared label. The default name is 'YourLabel.jpg'.
  3. Unlabeler Album : You can save the label to the 'TheUnlabeler' album. If the album does not exist, it will be created automatically.
  4. Unlabel Animation : After pressing the 'unlabel' button you will see an animation until your final label is calculated. Depending on the speed of your device and the size of the label the calculation is normally faster than the animation so you will see just the beginning. If you want to see the complete animation you can activate this setting.

These settings allow you to tailor The Unlabeler to your preferences, ensuring you capture the perfect label shot every time.
